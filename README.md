# 📊 Business Sales Performance Analytics Dashboard

## 📌 Project Overview
This project analyzes sales performance using the Superstore dataset. The goal is to uncover business insights by examining sales, profit, customer behavior, product performance, and regional trends. Python was used for data cleaning and exploratory data analysis (EDA), while Power BI was used to build an interactive dashboard.

---

## 🎯 Objectives
- Analyze overall sales and profit performance.
- Identify top-performing products and categories.
- Compare sales across regions and states.
- Understand customer segments and shipping preferences.
- Build an interactive dashboard for business decision-making.

## 📂 Dataset
**Dataset:** Sample Superstore Dataset

The dataset contains:
- 9,994 records
- 21 columns

Important columns include:
- Sales
- Profit
- Category
- Sub-Category
- Region
- State
- Customer Name
- Ship Mode
- Order Date

### Dashboard 2 – Interactive Dashboard
Includes slicers for:
- Region
- Category
- Segment

Interactive visuals:
- Sales by State
- Sales by Segment
- Sales by Ship Mode

---

## 📈 Key Insights
- Total Sales exceeded **2.30 Million**.
- Technology generated the highest sales.
- The West region recorded the highest revenue.
- California was the top-performing state.
- Consumer customers contributed the highest sales.
- Standard Class was the most frequently used shipping mode.
